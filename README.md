Адаптивная страница товара, разработанная по макету Figma. Реализована на **HTML5, CSS3 и Vanilla JavaScript**.
##  Особенности
- Полная адаптивность (мобильные, планшеты, десктоп)
- Модульная архитектура CSS
- Интерактивная галерея с превью
- Табы (описание, характеристики, отзывы, доставка)
- Карусель похожих товаров
- SEO-оптимизация (Schema.org, Open Graph)
- Доступность (ARIA-атрибуты)
- Производительность (lazy loading, preload)

## Запуск
Откройте `index.html` в браузере или через Live Server.

## Технологии
HTML5, CSS3 (Flexbox, Grid, Custom Properties), Vanilla JavaScript (ES6+)

## Лицензия
Проект создан в демонстрационных целях.
